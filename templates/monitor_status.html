About Folder Monitoring

The folder monitor automatically processes REFLIV log files as they're updated. Key features:

  • Continuous monitoring of specified folders
  • Incremental file processing (only reads new content)
  • Automatic file rotation (max 10 files)
  • Configurable polling intervals
  • Pattern-based file filtering
Configuration Tips:
  • Use *.txt,*.log for include patterns
  • Set polling interval based on file update frequency
  • Monitor will automatically start with active folders
  • Files are processed incrementally to avoid duplicates
